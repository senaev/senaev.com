Чтобы воспользоваться приложением, сгенерируйте массив из произвольных элементов, для этого либо впишите в поле ввода под словом 'count' желаемое количество случайных элементов и нажмите на кнопку 'generate', либо добавляйте по одному элементу с помощью формы ввода значеня под заголовком 'number' и кнопки 'add'. Когда массив будет создан, нажмите кнопку 'start', чтобы начать визуализацию процесса сортировки. По окончанию вы можете воспользоваться кнопкой 'finish', чтобы взглянуть на отсортированный и начальный массивы. Чтобы повторить процедуру, нажмите кнопку 'clear'. Обратите внимание, что 'start', 'finish' и 'clear' - по сути одна кнопка, которая в зависимости от потребностей меняет свою функциональность. Также существуют ограничения на элементы массива - их должно быть не более 20, значения - от -99 до 999.
